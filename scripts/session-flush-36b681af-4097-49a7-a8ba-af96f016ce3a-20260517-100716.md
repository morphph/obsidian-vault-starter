## 🎓 MODEL LITERACY

**Mixture of Experts (MoE) and Active Parameters**: Today's 30B-A3B reasoning model hits Olympiad gold using only 3B of its 30B parameters per forward pass. MoE architectures route each input to a small subset of specialized "expert" sub-networks rather than activating every parameter for every token. This slashes inference cost while maintaining the model's total knowledge capacity. It's why a model can be "30B" in stored knowledge but "3B" in compute per query — and why the parameter count on the label is increasingly misleading. When comparing models, ask about active parameters, not total parameters.
