# Raw Sources

Immutable source documents. Human curates what goes in. LLM reads but never modifies after creation.

## Conventions
- **Date-prefix format:** `YYYY-MM-DD-descriptive-title.md`
- **Immutable:** Never modify a file after initial creation. Corrections go in wiki page updates or new source files.
- **Accepted formats:** Articles, transcripts, paper notes, captured web pages, data files
- **Capture methods:**
  - Obsidian Web Clipper (browser extension → markdown)
  - WebFetch via `/ingest <url>`
  - Manual paste
- **Images:** Save to `raw/assets/`, reference from source files
