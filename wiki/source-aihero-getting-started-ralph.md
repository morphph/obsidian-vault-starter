# Source: Getting Started With Ralph

## Summary
Matt Pocock's step-by-step quickstart guide for the Ralph Wiggum technique. Prerequisites (Claude Code, Docker Desktop 4.50+), HITL script (`ralph-once.sh`) → AFK loop script (`afk-ralph.sh`), PRD creation, and configuration reference.

## Source Details
- **URL**: https://www.aihero.dev/getting-started-with-ralph
- **Author**: [[matt-pocock|Matt Pocock]] (AI Hero)
- **Type**: Tutorial / quickstart guide

## Key Claims
1. Prerequisites: Claude Code CLI + Docker Desktop 4.50+
2. `--permission-mode acceptEdits` prevents loop interruption
3. `-p` flag enables print mode for output capture
4. `<promise>COMPLETE</promise>` as standard termination signal
5. HITL script first, AFK script after trust built
