# PM OS

## Summary
AI-powered "operating system for product managers" 由 [[george-nurijanian|George]] 在 prodmgmt.world 运营。**v2 即将发布**（per 5/17 文章），带 improved memory + agents + skill workflows。定位：让 PM 工作有一个 operating layer —— memory / context / workflows / assumptions / research / decisions / measurement / review 都结构化，让 agent 有"比模糊 ticket 更好的东西可咀嚼"。

## Details

### 产品定位
- **问题**：PM 写 spec 是 tacit knowledge —— 每次都要从零思考 "什么 behavior 重要 / 哪些 edge cases 危险 / 什么 constraints load-bearing / 什么 validation evidence 算数 / 什么时候 stop agent 找人"
- **解法**：把这些 patterns 编码成 reusable workflows，每次新任务都从更锐利的 definition of done 开始
- **核心承诺**：让你不用通过"years of painful product reps"学到 PM craft

### 产品组件（per George 自己的描述）
- **External skills 集成**：内置一些他常用的 skill，如 Ryan Singer 的 Shaping skills
- **支持加 community skills**：George 提到正在用 Matt Pocock 的 `/grill-me`
- **V2 新特性**：improved memory + agents + skill workflows

### 商业模式
- 付费产品（链接 utm 显示是 lead-gen 模式）
- 推广 channel：X long-form articles（27K-46K views per article）+ 自家网站 + newsletter

### 与本 wiki 知识的关系
PM OS 本质是把 [[agent-ready-requirements]] + [[goal-template]] + [[mattpocock-skills-library]] 风格的 skill 集合**打包成给 PM 受众的开箱即用产品**。
- 跟 [[gbrain]] / [[gstack]]（Garry Tan 给 engineer 的开源版）对偶
- 跟 [[mattpocock-skills-library]]（Matt 给 engineer 的开源版）对偶
- **PM OS 是给 PM 的同类产品，但商业化（非开源）**

### 对 vfan 的相关性
PM OS 是一个**可借鉴的 productize template** —— 把"打包好的 AI workflows for 特定受众"作为付费产品的模式。你如果做 "Indie Creator's AI OS" 或 "Content Creator's Spec Library"，可以参考 PM OS 的：
- pricing tier 结构
- 营销 channel（X long-form articles）
- skill bundling 策略
- "memory loop" 命名（V2 主打概念）
