# Source: New Capabilities for Building Agents on the Anthropic API

## Summary
Anthropic announcement of four new beta API capabilities alongside Claude Managed Agents launch. Focuses on the companion features that make agents more practical: code execution, MCP connector, Files API, and extended prompt caching.

## Key Takeaways
- **Code Execution Tool** — sandboxed Python for data analysis/visualization. 50 free hours/org/day, then $0.05/hr.
- **MCP Connector** — connect to remote MCP servers without client code. API handles connection management, tool discovery, error handling.
- **Files API** — upload documents once, reference across conversations. Integrates with code execution.
- **Extended Prompt Caching** — 1-hour TTL (vs 5-min standard). Up to 90% cost reduction for long-running agents.
- All features in public beta, available to all API accounts.
